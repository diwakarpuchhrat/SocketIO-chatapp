About

Streamline is a web app I built with Python and Flask. It allows you to join rooms, send messages and chat with your friends.

This webapp uses SocketIO for real time messaging which is also the key feature of the app, Flask for routing, redirecting, socket functions and authentication using Bcrypt and Mongodb

For UI I used basic HTML, CSS and Javascript

Furthermore, this is a free and open source and contains no complex integration right now but it may get very advanced in the future.

The source code for the Project can be found on my Github
